# 공식 데이터 연계 메모

이 문서는 MVP에 어떤 공식 데이터가 들어갈 수 있는지 정리한 메모다.

## 1. 배경지도/공간 레이어
- 브이월드 WMS/WFS
- 배경지도, 행정경계, 각종 공간 레이어

## 2. 산불
- 국립산림과학원 산불위험예보정보
- 산림청 실시간 산불정보
- 산림청 산불상태별 이력
- 초기에 화선은 상황실 GeoJSON 업로드로 시작

## 3. 대피
- 행정안전부 민방위대피시설
- 행정안전부 이재민 임시주거시설

## 4. 물
- 행정안전부 비상급수시설
- 전국소방용수시설표준데이터
- 한국농어촌공사 저수지 정보/수위
- 한국수자원공사 지하수 관정정보

## 5. 산지
- 산림청 등산로 JSON/GPX/SHP
- 산사태 우려지역
- 산사태 위험지도(WMS)

## 연계 방식 제안

- **지도 표출:** 프런트에서 WMS 직접 오버레이
- **검색/경로 계산:** 백엔드에서 ETL 후 GeoJSON/DB 적재
- **화선:** 드론/상황실 업로드 우선, 이후 내부 피드 연계
